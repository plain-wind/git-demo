## 一、 git 命令

- 初始化 git 仓库

```bash
 git init
```

- 添加文件到暂存区

```bash
 git add .
```

- 提交暂存区文件到本地仓库

```bash
 git commit -m '提交信息'
```

- 查看提交日志

```bash
 git log
```

- 查看提交日志

```bash
 git reflog
```

- 回滚到指定提交

```bash
 git revert <commit-id>
```

- 回滚到指定提交

```bash
 git reset --hard <commit-id>
```

- 查看分支

```bash
 git branch
```

- 切换分支

```bash
 git checkout <branch-name>
```

- 合并分支

```bash
 git merge <branch-name>
```
